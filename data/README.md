# Data

Resultados de consultas SQL sobre la base de datos de viajes en taxi de Chicago
(noviembre 2017). A diferencia de otros proyectos, **estos CSV sí se incluyen**
porque son ligeros y permiten reproducir el análisis sin acceso a la base.

| Archivo | Contenido |
|---|---|
| `project_sql_result_01.csv` | Número de viajes por compañía de taxi. |
| `project_sql_result_04.csv` | Promedio de viajes por barrio de destino. |
| `project_sql_result_07.csv` | Viajes con su fecha/hora, clima y duración. |
